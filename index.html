What it shows: A stacked area chart where each colour band represents one clock-hour of the day. The combined height of a column equals the total number of events in that period.

How to read it: Hover over a column to see the exact count for every hour. When the Cumulative toggle is on, a red line on the right axis shows the running total of all events.

What to look for: Tall columns mark high-activity periods. The colour mix reveals which hours of the day drive the volume. A steepening cumulative slope signals acceleration; a flattening slope signals slowdown.

Activity heatmap

What it shows: A grid where each cell's darkness encodes the event count for a specific time slot within a period. X-axis = time period (day / week / month); Y-axis = sub-slot (hour of day / weekday / day of month).

How to read it: Darker cells = more events. Hover a cell to see the exact count and coordinates.

What to look for: Dark columns mark busy periods overall. Dark rows reveal time slots that are consistently active. Isolated dark cells are one-off spikes worth investigating.

Slot load distribution

What it shows: For each time slot (e.g. hour of day), this heatmap shows how often that slot reached a given event count across all periods in the selected range. X-axis = time slot; Y-axis = event count per slot; cell darkness = number of periods with that count.

How to read it: Hover a cell for exact figures. The Y-axis only includes count values that actually occurred — gaps in Y mean those counts were never observed.

What to look for: A tall dark cluster on a slot = predictable load. Scattered cells across a wide Y range = high variability. Blank slots = the period never had any events at that time.

Time between events

What it shows: A box-and-whisker plot of the gap (in hours) between consecutive events, grouped by the selected bucket period.

How to read it: The box spans the IQR (25th–75th percentile); the centre line is the median; whiskers extend to the full range; red dots are outliers. Hover for exact statistics.

What to look for: A tight, low box = events arrive at a consistent, frequent pace. A tall box or many outliers = irregular bursts with long quiet stretches in between. A rising median over time = events are becoming less frequent.
